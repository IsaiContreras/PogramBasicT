##MANUAL DE USUARIO
Bienvenido al manual de usuario. Antes de usar el programa, lea atentamente
este manual de uso y siga cada instrucción para manejar datos.
Este programa le facilitará el registro y la administración de sus alumnos.

##Navegación
Para navegar a través del menú, se desplegará en pantalla una lista de opciones,
para seleccionar solo presiona la tecla del número indicado para cada opción.

##Lista de alumnos 
Para ingresar a la lista de alumnos, al iniciar el programa presione la tecla "2".
Al ingresar, se mostrará la lista de alumnos:
La lista tiene capacidad para almacenar 20 espacios de alumnos.
Si existe un espacio ocupado, los datos del alumno se mostrarán en el
siguiente formato

-# Espacio-   -MATRÍCULA
              -APELLIDO Y NOMBRE
              -CORREO ELECTRÓNICO -TELÉFONO
              -DIRECCIÓN
              -CALIFICACIONES
              -PROMEDIO FINAL

Si existe un espacio desocupado o vacío, solo se mostrará el número de espacio
junto a la palabra --VACÍO--.

Debajo de la lista de alumnos, se desplegará las opciones de edición de lista.
Para seleccionar una opción, presione la tecla del número indicado para cada opción.

##Registrar alumno
Para registrar un alumno, selecciona la opción Registrar en la lista de alumno.
Una vez dentro, ingrese el número de espacio donde desea registrar al alumno e ingrese
los siguientes datos:

-Nombre : Si posee más de un nombre, ingrese los dos en orden habitual y separados por un espacio.
-Apellido Paterno.
-Apellido Materno.
-Matrícula.
-Correo Electrónico: Este debe cumplir con ciertos carácteres para ser validado.
     Debe contener un arroba, y terminar con ".com"
- Teléfono: No debe pasar de los 12 caracteres.
-Dirección:
    -Colonia
	-Calle
	-Número.

##Eliminar alumno
Pare eliminar un alumno del registro, ingrese a la opción Eliminar alumno debajo de la lista
de alumnos. Se desplegará la lista de alumnos para que pueda observar el número de espacio
de cada alumno. Para eliminar del registro, ingrese el número de espacio del alumno que
desea eliminar.

##Agregar calificación
El programa tiene espacio para guardar 3 calificaciones para cada uno de los alumnos registrados.
Para agregar una calificacion, ingrese a la opción Agregar calificación debajo de la lista de 
alumnos. A continuación ingrese el número de espacio del alumno. Especifique el espacio
de calificacón ingresando un número del 1 al 3 e ingrese la calificación. Recuerda que solo se admiten
valores de 0 a 100 para las calificaciones.
Se le preguntará si desea agregar otra calificación para el mismo alumno, si así lo desea presione "1".
Repita el proceso.
Se le preguntará si desea agregar calificación para otro alumno, si así lo desea presione "1".
Repita el proceso.

##Buscar y Mostrar
Esta función permine buscar un registro en base a la matrícula del alumno y mostrarlo en pantalla.
Para acceder a esta función, ingrese a la opción Buscar y Mostrar debajo de la lista de alumnos.
Para buscar un registro, ingrese la matrícula del alumno que quiere buscar.
Si el registro existe, se mostrara su información en pantalla.

##Buscar y Modificar.
Esta función permine buscar un registro en base a la matrícula del alumno y modificar los datos de este.
Para acceder a esta función, ingrese a la opción Buscar y Modificar debajo de la lista de alumnos.
Para buscar un registro, ingrese la matrícula del alumno que quiere buscar.
Si el registro existe, se mostrará su información actual en pantalla y le permitirá ingresar de nuevo
los datos del alumno. Modifique el campo que desea.

##Buscar y Eliminar
Esta función permine buscar un registro en base a la matrícula del alumno y eliminarlo.
Para acceder a esta función, ingrese a la opción Buscar y Eliminar debajo de la lista de alumnos.
Para buscar un registro, ingrese la matrícula del alumno que quiere buscar.
Si el registro existe, se mostrara su información en pantalla y se le preguntará si desea eliminar
ese registro. Si está seguro de eliminarlo presione "1".

##Guardar cambios
Si desea guardar los cambios ya hechos, ingrese a la opción Guardar cambios debajo de la lista de alumnos.
Se guardarán los datos en el disco.

##Salir del programa
Para salir del programa solo seleccione la opción Salir en el menú principal.
Al salir, se ejecutara la función de guardar cambios para asegurar sus cambios.
